# Object literal

- Open the file `person.js`{{open}}
- Declare an object `person` with the following properties :
  - name (string)
  - age (number)
- Print the variable `person` in your console.

Run `node person.js`{{execute}} to execute your code.

Example of expected result :

<pre class="file">
{
  name: "Paul",
  age: 32  
}
</pre>
